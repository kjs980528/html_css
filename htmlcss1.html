HTML

HTML(하이퍼텍스트 마크업 언어, HyperText Markup Language)는 웹 페이지를 만들 때 사용하는 마크업 언어야.
웹 브라우저가 HTML 코드를 해석해서 우리가 보는 웹 페이지를 화면에 표시하는 역할을 해.


CSS

CSS (Cascading Style Sheets, 캐스케이딩 스타일 시트)
CSS는 HTML로 만든 웹 페이지의 디자인과 스타일을 꾸며주는 언어야.
HTML이 웹 페이지의 뼈대(structure)를 만든다면, CSS는 그 뼈대에 색상, 레이아웃, 애니메이션 등 스타일을 적용하는 역할을 해.



JAVASCRIPT

**JavaScript(JS)**는 웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어야.
HTML이 웹 페이지의 구조, CSS가 디자인을 담당한다면, JavaScript는 인터랙션과 기능을 구현하는 역할을 해.
✅ 웹 브라우저에서 실행되는 대표적인 언어이며, ✅ 서버 개발(Node.js), 모바일 앱, 게임 개발 등 다양한 용도로 사용돼.
